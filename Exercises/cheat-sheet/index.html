HTML & CSS CHEAT SHEET

CSS TABLE

Selector Description Usage
letter-spacing

The letter-spacing property is used to set the horizontal spacing between the individual characters in an element. It’s not common to set the spacing between letters, but it can sometimes help the readability of certain fonts or styles. The letter-spacing property takes length values in units, such as 2px or 0.5em.

 "letter-spacing: 20px"
Word Spacing

You can set the space between words with the word-spacing  property. It’s also not common to increase the spacing between words, but it may help enhance the readability of bolded or enlarged text. The word-spacingproperty also takes length values in units, such as 3pxor 0.2em.

"word-spacing: 0.3em;"
Line height

We can use the `line-height` property to set how tall we want each line containing our text to be. Line height values can be a unitless number, such as `1.2`, or a length value, such as `12px`, `5%`or `2em`. Generally, the unitless value is preferred since it is responsive based on the current font size. In other words, if the `line-height`is specified by a unitless number, changing the font size will automatically readjust the line height.

line-height: 1.4;
Text-align

The text-align property, which you may already be familiar with from the CSS Visual Rules lesson, aligns text to its parent element.

 text-align: right;

HTML TABLE

TAG Description Usage
aside

The aside element represents a portion of a document whose content is only indirectly related to the document's main content. Asides are frequently presented as sidebars or call-out boxes.

 <aside> </aside>
Figure

The <figure> HTML element represents self-contained content, potentially with an optional caption, which is specified using the <figcaption> element. The figure,its caption, and its contents are referenced as a single unit.

<figure></figure>
Figcaption

The <figcaption> HTML element represents a caption or legend describing the rest of the contents of its parent <figure> element.

<figcaption></figcaption>
